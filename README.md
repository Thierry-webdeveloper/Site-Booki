# Site-Booki
Projet 3 de la formation OpenClassrooms "Intégrateur Web"

Mission : Créer la page d'accueil d'une agence de voyage à partir de maquettes Figma fournies (mobile, tablette et desktop), en utilisant uniquement HTML et CSS.
  - Créer ses propres composants d'interface pour correspondre fidèlement aux maquettes, tout en rendant le site responsive.
  - Le projet inclut également la mise en place d'un environnement de développement front-end.
